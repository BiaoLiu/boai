# 91小保微信公众号

## 简介
91小保是一款基于Django开发的面向个人代缴社保、公积金的微信公众号

## 功能特性
1. 微信登录
2. 微信公众号自定义菜单
3. 微信事件消息回复
4. 微信支付

## 相关技术
1. Python3.5+Django1.9+MySQL+Redis
